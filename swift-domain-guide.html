Setup Instructions
Connecting swiftlocating.ca
to Your New Website
This guide walks you through the exact steps to connect your GoDaddy domain to your new Swift Line Locating website. The whole process takes about 10 minutes on your end — we handle everything else.
Your Domain
swiftlocating.ca
Domain Registrar
GoDaddy
Est. Time
10 min setup · 1–24h DNS
1
Log Into GoDaddy and Open Your DNS Settings
You'll need to access the DNS management panel for swiftlocating.ca. Here's exactly how to get there:
1
Go to godaddy.com and sign in to your account
2
Click your name in the top right corner → select My Products
3
Find swiftlocating.ca in your list of domains
4
Click the DNS button next to the domain (or click the three dots → Manage DNS)
5
You'll land on the DNS Records page — this is where we'll make all the changes
2
Delete Any Existing A Records for "@"
GoDaddy adds default A records when you register a domain. These need to be removed before adding ours, or the domain won't point to the right place.
⚠️ Important: Look for any existing records with Type = A and Name = @. Delete all of them before continuing. Also delete any existing CNAME record with Name = www.
1
Scroll through the DNS records list and find rows where Type = A and Name = @
2
Click the trash icon or Delete next to each one
3
Also find and delete any CNAME record where Name = www
4
Leave everything else (MX records, TXT records) completely untouched
3
Add 4 New A Records (Root Domain)
These records point swiftlocating.ca (without www) to GitHub's servers. You need to add all four.
Add each of these as a separate A Record — click "Add New Record" for each one:
Type Name Value / Points To TTL
A@185.199.108.1531 Hour
A@185.199.109.1531 Hour
A@185.199.110.1531 Hour
A@185.199.111.1531 Hour
1
Click Add New Record (or the + button)
2
Set Type to A
3
Set Name to @
4
Paste the IP address into the Value or Points To field
5
Leave TTL at 1 Hour (default) and save
6
Repeat for all four IP addresses
4
Add 1 CNAME Record (www)
This ensures www.swiftlocating.ca also works and redirects properly.
Type Name Value / Points To TTL
CNAMEwwwjack108510.github.io1 Hour
1
Click Add New Record
2
Set Type to CNAME
3
Set Name to www
4
Set Value to jack108510.github.io
5
Save
